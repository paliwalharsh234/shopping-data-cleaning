# Shopping Dataset Cleaning using Pandas

## Objective
Perform data exploration and cleaning using Python and Pandas.

## Tasks Completed
- Loaded CSV dataset into Pandas DataFrame
- Explored dataset using head(), tail(), shape, columns, and info()
- Identified and handled missing values
- Filtered rows and selected columns
- Removed duplicate rows
- Created derived column
- Saved cleaned dataset into a new CSV file

## Technologies Used
- Python
- Pandas
- Jupyter Notebook

## Files Included
- shopping_data.csv
- shopping_data_cleaning.ipynb
- cleaned_shopping_data.csv
- README.md
